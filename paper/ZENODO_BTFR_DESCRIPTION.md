# Zenodo Upload: BTFR Evolution Paper

## Title
```
Evolution of the Baryonic Tully-Fisher Relation with Redshift: A Testable Prediction
```

## Creators
```
Zimmerman, Carl
```

## Description
```
We derive the evolution of the Baryonic Tully-Fisher Relation (BTFR) with redshift from the assumption that the MOND acceleration scale tracks cosmic density.

PREDICTION: At fixed rotation velocity, high-redshift galaxies have LESS baryonic mass:
Δlog M_bar = -log₁₀(E(z))

At z=2, this corresponds to a -0.48 dex offset — galaxies are 3× LESS massive at fixed v than the local BTFR predicts. This is counterintuitive (one might expect high-z galaxies to be more gas-rich) but follows directly from evolving a₀.

TEST: We compare against kinematic data from KMOS3D, SINS, and ALPINE surveys (10 galaxies, z=1.8-5.0):
• Evolving a₀: χ² = 118.7
• Constant a₀: χ² = 416.6
• Improvement: 3.5×

FALSIFICATION: This prediction is falsified if:
1. High-z BTFR shows no offset from local (within ±0.1 dex)
2. Offset is positive (more mass at fixed v)
3. Offset magnitude disagrees with -log₁₀(E(z))

NOT A FIT: The evolution E(z) is calculated from Planck cosmology (Ω_m=0.315, Ω_Λ=0.685). No parameters are fitted to the BTFR data.

Code: https://github.com/carlzimmerman/zimmerman-formula
```
