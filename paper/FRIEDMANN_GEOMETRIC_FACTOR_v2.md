## Appendix: Algebraic Identity

**Claim:** sqrt(3pi/2) = 4pi / 2sqrt(8pi/3)

**Proof:**

    4pi / 2sqrt(8pi/3)
    = 2pi / sqrt(8pi/3)
    = 2pi * sqrt(3/(8pi))
    = 2pi * sqrt(3) / sqrt(8pi)
    = 2pi * sqrt(3) / (2*sqrt(2)*sqrt(pi))
    = pi * sqrt(3) / (sqrt(2)*sqrt(pi))
    = sqrt(pi) * sqrt(3) / sqrt(2)
    = sqrt(3pi/2)  QED
