# Zenodo Upload: The Hubble Constant from Galaxy Dynamics

## Title
The Hubble Constant from Galaxy Dynamics: An Independent Measurement Using the MOND Acceleration Scale

## Authors
Carl Zimmerman

## Description (for Zenodo abstract field)

We present a novel, independent measurement of the Hubble constant H₀ using the MOND acceleration scale a₀ measured from galaxy rotation curves.

**Method:**
From the Zimmerman Formula a₀ = cH₀/5.79 (where the coefficient 5.79 = 2√(8π/3) is derived from the Friedmann equation), we invert to obtain:

H₀ = 5.79 × a₀ / c

**Input:**
The MOND acceleration scale a₀ = (1.20 ± 0.02) × 10⁻¹⁰ m/s², as measured from the Radial Acceleration Relation in 175+ disk galaxies (McGaugh et al. 2016, SPARC database).

**Result:**
H₀ = 71.5 ± 1.2 km/s/Mpc

**Key finding:**
This value lies between Planck (67.4 ± 0.5) and SH0ES (73.04 ± 1.04), with:
- 3.2σ tension with Planck CMB
- 0.9σ agreement with SH0ES Cepheids
- 0.1σ agreement with GW170817 standard siren

**Independence:**
This measurement uses neither the CMB nor the astronomical distance ladder. It relies solely on galaxy kinematics — rotation curves and velocity dispersions — making it completely independent of both Planck and SH0ES systematic errors.

**Verification:**
The coefficient is empirically validated: using Planck's ρ_c, the formula a₀ = c√(Gρ_c)/2 predicts a₀ = 1.19 × 10⁻¹⁰ m/s², matching the observed value within <1%.

**Reproducibility:**
All calculations available at: https://github.com/carlzimmerman/zimmerman-formula/tree/main/paper/analysis
