This folder contains research that is **NOT** first-principles physics.

These files explore interesting patterns and ideas, but they should **NOT** be presented as derived results.

## Important Warning

**None of these files should be cited as derivations.**

They are explorations of where Z² patterns might appear, but finding Z² in a number is NOT the same as deriving that number from Z².

## What Would Make These Valid

For biology claims to become valid physics, we would need to:
1. Derive WHY protein angles are related to Z
2. Show a physical mechanism connecting geometry to biochemistry
3. Make testable predictions that distinguish Z² from chance

Currently, we have none of these.
