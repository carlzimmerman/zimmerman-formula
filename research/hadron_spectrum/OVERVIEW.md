## Notable Results

### Kaon/Pion Ratio
```
m_K/m_π = Z - 9/4 = 5.79 - 2.25 = 3.54
```
**0.03% accuracy** — remarkably precise

### Lambda/Proton Ratio
```
m_Λ/m_p = 1 + 3Ω_m/5 = 1 + 3(0.315)/5 = 1.189
```
This connects baryon masses to cosmological matter fraction!

### Omega/Proton Ratio
```
m_Ω/m_p = Z - 4 = 5.79 - 4 = 1.79
```

## Physical Significance

These ratios suggest hadron masses may be determined by:
1. **Geometric factors** (Z from cosmology)
2. **Simple corrections** (integers, simple fractions)
