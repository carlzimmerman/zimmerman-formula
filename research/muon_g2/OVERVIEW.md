# Muon g-2 Anomaly

## The Problem

The muon anomalous magnetic moment shows a discrepancy between experiment and Standard Model prediction:

```
Δa_μ = a_μ(exp) - a_μ(SM) ≈ (251 ± 59) × 10⁻¹¹

This is a ~4.2σ discrepancy (as of Fermilab 2023)
```

## Zimmerman Exploration

This directory explores whether the muon g-2 anomaly can be connected to Z.

### Potential Formula
```
Δa_μ ≈ α²/(πZ) or similar structure
```

### Current Status

**SPECULATIVE** — No confirmed formula yet. This is exploratory research.

## Why This Could Matter

The muon g-2 anomaly is sensitive to:
- New physics beyond the Standard Model
- Virtual particle loops
- Possible gravitational corrections

If the anomaly involves Z, it would suggest:
- Connection between muon magnetic moment and cosmology
- A new source of loop corrections involving spacetime geometry
