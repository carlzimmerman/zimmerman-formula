## How to Interpret Kd Values

All Kd values in this repository (e.g., "0.01 nM", "0.25 nM") are:

- **NOT** physics-based predictions
- **NOT** experimentally measured
- **NOT** from FEP/MD simulations

They ARE:

- Heuristic scores based on sequence features
- Useful for **ranking** candidates
- Starting points for **experimental validation**

**Do not cite these as "predicted binding affinities"**
