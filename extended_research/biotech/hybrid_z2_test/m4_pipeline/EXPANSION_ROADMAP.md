# Open Therapeutic Sequence Project: Expansion Roadmap

## Philosophy

This roadmap identifies **scientifically validated** computational approaches that an
independent researcher can execute using:
- Public domain sequences (expired patents, PDB, UniProt, GenBank)
- Peer-reviewed engineering techniques
- Open-source computational tools
- Defensive publishing for prior art establishment

**All approaches below use established biochemistry, not theoretical frameworks.**

## IMPLEMENTATION PRIORITY

### Phase 1: Low-Hanging Fruit (Do First)
1. **Expired patent antibodies** (Humira, Herceptin, Avastin, Rituxan)
   - Sequences fully public
   - Add BBB peptides for CNS variants
   - Supercharge for stability

2. **More migraine antibodies** (you have 4, expand to include variants)

3. **Lysosomal enzymes + BBB peptides**
   - Huge unmet need for CNS forms of LSDs
   - Sequences public (UniProt)

### Phase 2: Moderate Complexity
4. **Cytokines with half-life extension**
5. **Bispecific formats** of existing antibodies
6. **Tumor-penetrating peptide fusions**

### Phase 3: Advanced
7. **De novo designed binders** (using RFdiffusion)
8. **Consensus-stabilized variants**
9. **Deimmunized versions**
