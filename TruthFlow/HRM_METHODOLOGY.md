# HRM: Honesty, Rigor, Methodology

## The TruthFlow Validation Philosophy

**One Rule:** We only trust what we can verify against real measurements from peer-reviewed sources.

---

## The Three Pillars

### H - HONESTY

**What we actually derived:**
- Z² = 32π/3 (definitional axiom)
- MOND scale a₀ = cH₀/Z (one genuine derivation)
- Cube geometry: 8 vertices, 12 edges, 6 faces (geometric fact)

**What matches but isn't derived:**
- α⁻¹ = 4Z² + 3 ≈ 137 (WHY 4 and 3?)
- sin²θ_W = 3/13 ≈ 0.231 (WHY these numbers?)
- Hierarchy = 2×Z^(43/2) (coefficient decomposition, not derivation)

**What is speculation:**
- String theory connections (26 = 8+12+6)
- Consciousness claims
- Biology claims

**Rule:** Every prediction must be labeled: DERIVED | MATCHES | SPECULATION

---

### R - RIGOR

**Source Requirements:**

| Tier | Source Type | Examples |
|------|-------------|----------|
| 1 | Official databases | CODATA, PDG, Planck Collaboration |
| 2 | Peer-reviewed papers | arXiv with journal publication |
| 3 | Preprints | arXiv without peer review |
| 4 | Community data | SPARC, THINGS, surveys |

**Citation Format:**
```
Value: 0.685 ± 0.007
Source: Planck 2020 PR4
Paper: arXiv:1807.06209
Table: Table 2, Column 5
Accessed: 2026-05-02
```

**Validation Thresholds:**
- σ < 2: VALIDATED (consistent)
- 2 ≤ σ < 3: TENSION (investigate)
- σ ≥ 3: FAILED (document and learn)

---

### M - METHODOLOGY

**The Pipeline:**

```
┌─────────────────────────────────────────────────────────────┐
│  1. FETCH                                                   │
│     └── Source: arXiv, NASA ADS, PDG, CODATA               │
│     └── Output: Raw papers/data with full citations        │
├─────────────────────────────────────────────────────────────┤
│  2. PARSE                                                   │
│     └── Extract: ONLY empirical values (not theory)        │
│     └── Filter: Keywords "measured", "observed", "found"   │
│     └── Reject: "predict", "model", "assume"               │
├─────────────────────────────────────────────────────────────┤
│  3. COMPUTE                                                 │
│     └── Input: Z² = 32π/3 (ONLY this)                      │
│     └── Output: Predictions with NO free parameters        │
│     └── Lock: Values cannot be adjusted post-hoc           │
├─────────────────────────────────────────────────────────────┤
│  4. ASSESS                                                  │
│     └── Compare: Z² prediction vs empirical value          │
│     └── Compute: σ tension                                 │
│     └── Classify: VALIDATED / TENSION / FAILED             │
├─────────────────────────────────────────────────────────────┤
│  5. LEARN                                                   │
│     └── Success: → validated_truths/                       │
│     └── Failure: → failed_attempts/ + root cause           │
│     └── Update: learnings/ with what we discovered         │
└─────────────────────────────────────────────────────────────┘
```

## The Honesty Matrix

For each Z² prediction, we must answer:

| Question | Answer Options |
|----------|----------------|
| Is this derived from first principles? | YES / PARTIAL / NO |
| Is the derivation mechanism known? | YES / UNKNOWN |
| Does it match empirical data? | YES (σ<2) / TENSION (2-3σ) / NO (σ>3) |
| Can it be falsified? | YES (specific test) / NO |
| Source quality? | TIER 1-4 |
