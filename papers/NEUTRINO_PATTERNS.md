## Overview

## Part 1: The Neutrino Data

### Mass Squared Differences

```
Δm²₂₁ = m²₂ - m²₁ = 7.53 × 10⁻⁵ eV²  (solar)
Δm²₃₁ = |m²₃ - m²₁| = 2.51 × 10⁻³ eV²  (atmospheric)
```

**Ratio:**
```
Δm²₃₁/Δm²₂₁ = 33.3
```

### Mixing Angles

```
sin²θ₁₂ = 0.304 ± 0.012  (solar)
sin²θ₂₃ = 0.573 ± 0.016  (atmospheric)
sin²θ₁₃ = 0.0222 ± 0.0007  (reactor)
```

### CP Phase

```
δ_CP ≈ 195° (poorly constrained)
```

## Part 5: Neutrino Mass Hierarchy

### The Mass Scale

Absolute neutrino masses are unknown, but:
```
Σm_ν < 0.12 eV (cosmological bound)
m_lightest > 0 (at least one non-zero)
```

### Possible Z Connection

**If:**
```
m_ν ∝ (some scale) / Z^n
```

With eV-scale ~ electron mass:
```
m_ν ~ m_e / (4Z²) ~ 0.5 MeV / 134 ~ 4 meV
```

This is roughly the right scale! (m_ν ~ 10-50 meV)

**More precise:**
```
Σm_ν = m_e / (4Z² + 3) = 0.511 MeV / 137 = 3.7 meV × 3 ~ 0.01 eV
```

Order of magnitude correct.

### The Hierarchy

**Normal hierarchy (NH):** m₁ < m₂ << m₃
**Inverted hierarchy (IH):** m₃ << m₁ < m₂

**Zimmerman prediction?**
If Δm²₃₁/Δm²₂₁ = Z², this favors NH (the ratio is large).

## Part 6: The See-Saw Connection

### Standard See-Saw

```
m_ν = m_D^T M_R⁻¹ m_D
```

Where:
- m_D = Dirac mass (electroweak scale)
- M_R = Majorana mass (GUT scale)

### Z in See-Saw

If M_R is related to Z:
```
M_R = M_GUT × f(Z)
```

Then:
```
m_ν = v² / M_R = v² / (M_GUT × f(Z))
```

The light neutrino masses would depend on Z.

### Specific Model

This is close to Σm_ν < 0.12 eV!
